# Лабораторная работа №7 — jQuery

Сравнение jQuery и современного Vanilla JavaScript.

### Что реализовано:
- Вкладки (Tabs) — jQuery
- Плавная прокрутка к секциям — jQuery + Vanilla
- Загрузка случайной цитаты через AJAX
- Перетаскиваемый блок и календарь через jQuery UI
- Современный дизайн + переключение тёмной/светлой темы

### Использованные функции jQuery:
`.click()`, `.addClass()/removeClass()`, `.animate()`, `$.get()`, `.draggable()`, `.datepicker()`

### Вывод:
jQuery был крут в 2006–2015, но сегодня почти всё можно сделать проще и быстрее на чистом JS.  
Для новых проектов — только Vanilla JS / React / Vue.  
Но знать jQuery нужно — 70% сайтов до сих пор на нём!
